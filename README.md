# 🔥 Fire Detection using YOLOv11

This project is an AI-powered system designed to detect fires in photos,videos or in real-time using wencam. It utilizes YOLOv11 model, which is highly efficient for object detection tasks. 

The application is built with Streamlit, providing an interactive web interface where user can upload images or video streams to detect potential fire threats immediately.

## Model Training & Data
- The model was trained on a big, cleaned dataset of fire and smoke images for high precision.
- Performed extensive pre-processing and augmentation to ensure reliability in various environments.

## 🛠️ Tech Stack
- Model: YOLOv11 (Ultralytics)
- Framework: Streamlit
- Language: Python
- Computer Vision: OpenCV
